#格式字符串
##初步
|||
|:---:|:---:|
|%s|字符串|
|%r|字符串|
|%d|十进制整数|
|%x|十六进制整数|
|%o|八进制整数|
|%f|十进制浮点数|
|%e|科学计数法|
|%g|十进制或者科学计数法浮点数|
|%%|文本%本身|

__%r和%s的区别__:
- %r显示变量的原始数据
- %s向用户显示输出

```
>>>print("%r,%r" % ("one", "two"))
one, two
>>>print("%r, %r" % ("one", "two"))
'one', 'two'
``` 


##python3中的新的方式
__使用{}和format__
```
'{} {} {}'.format(n, f, s)
```

- 可以指定插入顺序
- 参数可以是字典或者变量名
